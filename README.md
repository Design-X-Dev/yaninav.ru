# ЯНИНА В - Ювелирная студия

Современный веб-сайт ювелирной студии, созданный на основе Next.js 15 с использованием Tailwind CSS.

## 🌟 Особенности

- **Современный дизайн** в стиле luxury с золотой цветовой схемой
- **Адаптивная верстка** для всех устройств
- **SEO-оптимизированный** с правильными метатегами
- **Высокая производительность** благодаря Next.js
- **Интерактивные компоненты** с плавными анимациями

## 🏗️ Структура сайта

### Основные секции:
1. **Header** - Навигация с логотипом и меню
2. **Hero** - Главная секция с призывом к действию
3. **Catalog** - Каталог украшений с фильтрацией
4. **About** - Информация о студии с ключевыми преимуществами
5. **Contact** - Контактная информация и форма обратной связи
6. **Footer** - Дополнительная информация и ссылки

### Каталог включает:
- Помолвочные кольца
- Обручальные кольца  
- Колье и цепи
- Серьги
- Браслеты
- Свадебная коллекция

## 🛠️ Технологии

- **Next.js 15** - React фреймворк
- **TypeScript** - Типизированный JavaScript
- **Tailwind CSS 4** - Utility-first CSS фреймворк
- **Google Fonts** - Playfair Display и Inter
- **React Hooks** - Для управления состоянием

## 🚀 Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
cd yaninav.ru
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер разработки:
```bash
npm run dev
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## 📝 Скрипты

- `npm run dev` - Запуск сервера разработки с Turbopack
- `npm run build` - Сборка для продакшена
- `npm run start` - Запуск продакшн сервера
- `npm run lint` - Проверка кода с ESLint

## 🔧 Кастомизация

### Изменение цветов:
Отредактируйте CSS переменные в `src/app/globals.css`:

```css
:root {
  --gold: #d4af37;
  --gold-light: #f7e98e;
  --gold-dark: #b8860b;
  /* ... другие цвета */}
```

### Добавление новых товаров:
Обновите массив `products` в компоненте `src/components/Catalog.tsx`

### Изменение контактной информации:
Отредактируйте компонент `src/components/Contact.tsx`
